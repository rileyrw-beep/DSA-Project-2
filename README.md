This is a project that compares the data structures: B+ Tree, Skip List, and Hash Table.

It uses the Mersenne Twister random number generator to propagate the 100,000 integer data points.

When ran in the command line users can run ".help" to see all the commands they can possibly do.

This includes .exit, .switch, insert, remove, and search.

The program starts on a B+ Tree which users can insert, remove, and search from.

Using .switch, users can switch to the next data structure.

With .exit users can end the program.
